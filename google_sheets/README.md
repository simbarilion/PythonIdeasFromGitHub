## Описание решения

### Используемый API

- В проекте используется публичный GitHub Search API:

**https://api.github.com/search/repositories**

- API используется для поиска популярных публичных Python-репозиториев.

### Используемый триггер

- В проекте реализован trigger onOpen().

- При открытии Google Таблицы автоматически создаётся пользовательское меню:

**GitHub Loader → Load repos**

- Нажатие на пункт меню запускает функцию загрузки данных из GitHub API.

### Логика обработки данных

#### Скрипт:

- Выполняет HTTP GET-запрос через UrlFetchApp.fetch
- Получает JSON-ответ от GitHub API
- Извлекает данные о репозиториях:
    - название
    - количество stars
    - количество forks
    - URL репозитория
- Записывает данные в Google Таблицу
- Обрабатывает ошибки API и сети

### Обработка ошибок

- Реализована базовая обработка ошибок:

    - ошибки API (statusCode != 200)
    - сетевые ошибки (try/catch)

- Статус выполнения и сообщения об ошибках записываются в отдельные ячейки таблицы.
